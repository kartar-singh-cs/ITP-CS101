# 📊 Student Grade Management System

A console-based student records system written in C++. Add students with marks for 3 subjects, view all records, search by name, and calculate the class average — with automatic grade assignment.

## Features
- Add multiple students with marks for 3 subjects
- Auto-calculates total, average, and letter grade (A / B / C / F)
- View all student records in a formatted table
- Search for a specific student by name
- Calculate and display overall class average with grade

## Concepts Used
- Arrays for storing student data
- Loops and switch-case
- Conditional logic for grade assignment
- Formatted output with `iomanip`

## How to Run
```bash
g++ -std=c++17 Grade-Management.cpp -o grade-management
./grade-management
```

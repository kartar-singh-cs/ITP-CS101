# 💸 Personal Expense Tracker

A console-based expense tracking system written in C++. Users can log expenses, view all records, calculate total and category-wise spending, and find the highest expense.

## Features
- Add expenses with title, category, amount, and date
- View all recorded expenses in a formatted table
- Calculate total spending across all entries
- Filter spending by category (Food / Travel / Shopping / Other)
- Find the single highest expense

## Concepts Used
- Vectors for dynamic data storage
- Functions with pass-by-reference
- Loops and switch-case
- Formatted output with `iomanip`

## How to Run
```bash
g++ -std=c++17 Expense-Tracker.cpp -o expense-tracker
./expense-tracker
```

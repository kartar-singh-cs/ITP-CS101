# 🚗 Car Rental System

A console-based car rental system written in C++. Users can view available cars, rent one, and return it — with availability tracking across the session.

## Features
- View all cars with rent per day and availability status
- Rent a car (marks it as unavailable)
- Return a car (marks it as available again)
- Input validation for all choices

## Concepts Used
- Arrays (strings, integers, booleans)
- Loops and switch-case
- Boolean logic for availability tracking
- Formatted output with `iomanip`

## How to Run
```bash
g++ -std=c++17 Car-Rental.cpp -o car-rental
./car-rental
```
